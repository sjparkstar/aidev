# 프로젝트 소개

## 🎯 프로젝트 목표

RemoteCall 관리자 화면의 UI/UX를 참고하여 Redmine API와 연동되는 게시판 시스템을 개발합니다.

## 🖼️ 주요 특징

### UI/UX 특징
- **RemoteCall v8.0 User Admin** 인터페이스 완전 재현
- 남색(slate-800) 접이식 사이드바
- 깔끔한 테이블 기반 데이터 표시
- 완전 반응형 디자인

### 기능적 특징
- **실시간 Redmine API 연동**
- **로드맵 및 버전 관리**
- **이슈 목록 표시**
- **다국어 지원** (한국어/영어/일본어)
- **검색 및 필터링**

### 기술적 특징
- **Next.js 15** App Router 활용
- **TypeScript** 타입 안전성
- **Tailwind CSS** 유틸리티 스타일링
- **React Context** 상태 관리

## 🚀 개발 배경

1. **기존 Redmine 웹 UI의 한계**
   - 구식 디자인
   - 모바일 반응성 부족
   - UX 개선 필요

2. **RemoteCall UI의 장점**
   - 현대적인 디자인
   - 직관적인 네비게이션
   - 관리자 친화적 구조

3. **목표**
   - 두 시스템의 장점 결합
   - 효율적인 프로젝트 관리
   - 개발팀 생산성 향상
